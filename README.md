# PHP Property Reader 

[![Build Status](https://github.com/ScrumWorks/property-reader/workflows/build/badge.svg)](https://github.com/ScrumWorks/property-reader)

## Installation
```
composer require amateri/property-reader
```

## Testing
You can run the tests with:

```
vendor/bin/phpunit
```
